⓪Gmailを開いて、GitHubの確認ボタンを押してください。

①以下フォルダに移動する
C:\xampp\htdocs

②GitBushを開く

③GitBushに以下コマンドをコピペ＆ペースト（ディレクトリの移動）
cd /c/xampp/htdocs/

④リモートリポジトリの内容をローカルリポジトリに取得する
git clone git@github.com:intecs2510ap03/popcorns.git
※C:\xampp\htdocs\popcornsのフォルダができたこと、
中身が入っていることを確認する

⑤GitBushに以下コマンドを入力する（ディレクトリの移動）
cd /c/xampp/htdocs/popcorns

⑥GitBushに以下コマンドを入力し、ファイルが入っていることを確認
ll(小文字のエルが２つです）


⑦ブランチを作成する

★平さん担当分
git branch index
git branch userAdd

★中さん担当分
git branch questions

★Tさん担当分
git branch questionInput
git function

★大さん担当分
git answer

★山げん担当分
git detail


⑧作成したブランチの一覧を表示する
git branch

⑨ブランチをスイッチする
git switch (自分が作ったブランチ名）
例：山げんなら「git switch detail」

⑩ブランチ一覧を表示して変更したことを確認する
git branch

⑪ファイルの中身を変更してみる
例：山げんなら「detail.php」を開き
　以下のように変更する（４行目）
　修正前：<?php echo 'detail.php'; ?>
　修正前：<?php echo 'detail　テスト'; ?>

⑫変更したファイルを保存して閉じる

⑬GitBushに以下コマンドを入力し、ステータスを確認する
git status

⑫変更内容をローカルリポジトリにコミットする
git commit -m 'ファイル変更とコミットの練習'

⑬作成したブランチをリモートリポジトリに反映する
git push origin (自分が作ったブランチ名）
山げんなら、git push origin detail
